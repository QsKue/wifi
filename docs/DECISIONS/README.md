# Decisions (ADRs)

Durable architectural decisions for `wifi`. One file per decision, numbered. An ADR records *why*
a choice was made so it isn't re-litigated; it is not a task log (git is).

- [0001-initial-architecture.md](0001-initial-architecture.md) — starting architecture: OS-native
  backends, compile-time selection, async API, phased rollout.

New decision: copy [decision-template.md](decision-template.md), give it the next number, and link
it here.
